# Yu Zhang (张宇) — Personal Academic Website

This is the source code for my personal academic website, hosted at [uni-zhang.github.io](https://uni-zhang.github.io).

## About Me

I am a **PhD Candidate** in Transportation Planning and Management at the **School of Economics and Management, Beihang University (北京航空航天大学)**, Beijing, China. I was a Visiting PhD at **KTH Royal Institute of Technology**, Stockholm, Sweden (2024–2025).

My research focuses on large-scale real-time on-demand mobility systems (e.g., ride-hailing, ride-sharing), with particular interests in dynamic ride-matching modeling, matching algorithm design, demand forecasting, mechanism design, and multi-modal intelligent transportation systems (e.g., integration of ground-based ride-hailing and urban air mobility).

## Site Structure

## Local Development

1. Install dependencies: `gem install github-pages`
2. Serve locally: `jekyll serve`
3. View at: [http://127.0.0.1:4000](http://127.0.0.1:4000)
